# ai-validator

[![npm downloads](https://img.shields.io/npm/dm/ai-validator.svg)](https://www.npmjs.com/package/ai-validator)

Generate Zod and Yup validation schemas from TypeScript types

## Install

```bash
npm install -g ai-validator
```

## Usage

```bash
npx ai-validator ./path/to/file
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
